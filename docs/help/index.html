Introduction

Bienvenue dans Homelet ! Cette application a été conçue pour simplifier l'apprentissage des textes pour les acteurs. Elle vous permet d'enregistrer, partager et répéter vos répliques en toute simplicité.

Premiers Pas

Pour commencer à utiliser Homelet, suivez ces étapes :

  • Installation : Téléchargez l'application depuis [Android/iOS Store].
  • Création d'un Compte : Connectez-vous ou inscrivez-vous pour accéder à toutes les fonctionnalités.
  • Création de Projets : Créez votre premier projet pour organiser vos répliques par scène.

Fonctionnalités Principales

1. Organisation des Projets

Chaque pièce ou œuvre est structurée en projets. Voici comment les organiser :

  • Ajouter un Projet : Appuyez sur le bouton "Créer un Projet".
  • Gestion des Scènes : Ajoutez des scènes à vos projets.
  • Attribution des Répliques : Associez chaque réplique à un personnage.

2. Enregistrement Audio

Enregistrez les répliques directement dans l'application :

  • Accédez à une scène et sélectionnez "Enregistrer".
  • Parlez dans le micro et attribuez la réplique au personnage concerné.
Astuce : Utilisez un casque pour une meilleure qualité d'enregistrement.

3. Mode Lecture

Le mode lecture vous permet de répéter vos répliques :

  • Lancez la lecture d'une scène.
  • Les répliques de votre personnage seront mutées automatiquement.
  • Réglez les paramètres de lecture pour ajuster les temps de pause.

Questions Fréquentes

1. Comment partager mes répliques avec d'autres acteurs ?

Accédez à un projet et utilisez l'option "Partager". Vous pouvez envoyer vos fichiers audio via e-mail ou messagerie instantanée.

2. Est-il possible d'utiliser Homelet sans connexion Internet ?

Oui, toutes les fonctionnalités principales fonctionnent hors ligne, mais la synchronisation et le partage nécessitent une connexion Internet.
